## üöÄ Quick Start

### Installation

1. Download the latest release from [Releases](https://github.com/Rayamit1010/ai-summary-buttons/releases)
2. Upload to WordPress: `Plugins ‚Üí Add New ‚Üí Upload Plugin`
3. Activate the plugin
4. Go to `AI Buttons ‚Üí Customize Buttons` to configure

### Basic Usage

```php
[ai_summary_buttons]
```

With custom label:
```php
[ai_summary_buttons label="Summarize with AI:"]
```

Specific buttons only:
```php
[ai_summary_buttons buttons="chatgpt,claude,gemini"]
```
