üíª Shortcode Usage

Basic Usage

[ai_summary_buttons]

Displays all enabled AI buttons with default settings.

With Custom Label

[ai_summary_buttons label="Try AI summaries:"]

Adds a custom label above the buttons.

Specific Buttons Only

[ai_summary_buttons buttons="chatgpt,claude,gemini"]

Shows only the specified AI services.

Combined Options

[ai_summary_buttons label="Get AI help:" buttons="chatgpt,perplexity"]

Combines custom label with specific buttons.
